Sistema simples desenvolvido em Python para calcular o **IMC (Índice de Massa Corporal)**, classificar o resultado e fornecer dicas básicas de saúde.

---

* Inserir peso e altura

* Exibir dicas de saúde
* Armazenar histórico durante a execução

---

## 📊 O que é IMC?

O IMC (Índice de Massa Corporal) é uma medida utilizada para avaliar se uma pessoa está dentro do peso ideal.

Fórmula:

IMC = peso / (altura x altura)

Classificação:

* Abaixo de 18.5 → Abaixo do peso
* 18.5 até 24.9 → Peso normal
* 25 até 29.9 → Sobrepeso
* 30 ou mais → Obesidade
